## 1. 前提と対象範囲（要約）
- 目的: 企業内利用のスケジュール管理Webアプリの新規開発とAzureコンテナ基盤へのデプロイ
- 技術候補: 
  - フロントエンド: React(TypeScript) + Vite/Next.js（SPA）
  - バックエンド: Node.js(Express/Fastify) または .NET Minimal API
  - DB: Azure Database for PostgreSQL (Flexible Server)
  - 認証/認可: アプリ内ローカル認証 + 役割ベースRBAC（SSOはオプション）
  - デプロイ: Azure Container Apps（または AKS はスコープ外）
  - IaC/CI: Bicep/Terraform + GitHub Actions
- ユースケース（抜粋）: 
  - カレンダー（月/週/日）表示、イベントCRUD、繰返し予定、参加者、基本的な検索/フィルタ
  - 管理者GUIで利用ユーザ・ロール・基本設定（営業日/休日・通知既定値 等）を操作
  - 監査ログ（重要操作）／エクスポート（CSV）
- 非機能: 初期ユーザ規模 200名、同時接続 50程度、RPO=1日/ RTO=数時間の目安、可用性SLAはAzure標準

## 2. スコープ（機能一覧）
- アプリ利用
  - カレンダーUI（月/週/日、ドラッグ&ドロップ操作、タイムゾーン/祝日対応）
  - 予定の作成/編集/削除、繰返し、参加者、予定の公開範囲（自分/チーム/全体）
  - 検索/フィルタ（期間・作成者・キーワード）
- サーバ一括管理
  - REST API（予定/ユーザ/設定）、DBスキーマ、共通バリデーション
  - 基本的な監査ログ（予定作成/削除、権限変更）
- 管理者GUI（設定ツール）
  - ユーザ/ロール管理（一般/管理者）
  - 営業日/休日・通知既定値・部署/チーム設定
- ブラウザアクセス
  - SPA + API構成、レスポンシブ対応
- Azureコンテナ
  - コンテナ化（Dockerfile）、ステージング/本番環境、Azure Container Appsへのデプロイ
  - IaC（Bicep/Terraform）とGitHub ActionsでCI/CD

スコープ外（本見積外/要追加見積）
- SSO（Azure AD 等）、外部グループウェア/通知（メール/SMS/Teams）連携、本格的な承認ワークフロー、多言語化、高可用クラスタ/DR設計、アプリ内チャット

## 3. 見積方式と係数（PARAMETERS.md 準拠）
- 粗利率: 35%
- リスク費率: 10%
- 単価（月額）: PM 1,200,000 円、エンジニア 900,000 円、QA 600,000 円
- ブレンド単価（参考・要合意）: 885,000 円/人月
- 算定方針: 機能別WBSの人月積上げ + リスク10% を原価に加算後、粗利35%を考慮

参考式:
- 原価（見積）= ブレンド単価 × 人月合計
- リスク加算後原価 = 原価 × (1 + 0.10)
- 提案価格（税別）= リスク加算後原価 ÷ (1 - 0.35)

## 4. 工数見積（人月）
- 要件定義/基本設計: 1.8
- 画面設計/UX設計: 0.6
- フロントエンド実装（カレンダー/予定CRUD/検索）: 3.2
- バックエンドAPI/認可/監査ログ: 2.4
- DB設計/移行（初期データ/祝日）: 0.6
- 管理者GUI（ユーザ/ロール/基本設定）: 1.2
- インフラ（Docker化/ACA/IaC/監視）: 1.0
- テスト（単体/統合/UAT支援、テスト設計含む）: 1.2
- PM/品質保証/ドキュメント: 0.9
小計: 12.9 人月

調整・余力（バッファ/仕様ブレ余地 10%）: +1.3 人月
合計: 14.2 人月

注: 実装詳細や既存資産の有無により±20%程度の変動可能性があります。

## 5. 体制・スケジュール（提案）
- 期間目安: 4〜5ヶ月
- 体制（ピーク時）: PM×1, FE×2, BE×2, QA×1（兼務含む）
- マイルストーン（例）
  - W1-3: 要件定義/基本設計
  - W4-7: 画面/API詳細設計・プロトタイプ
  - W8-13: 実装/単体/統合、IaC/CI整備
  - W14-17: 総合テスト/UAT対応
  - W18-20: 本番リリース/運用引継

## 6. 金額試算（税別）
前提: ブレンド単価 885,000 円/人月、リスク10%、粗利35%
- 人月: 14.2 人月
- 原価概算: 14.2 × 885,000 = 12,567,000 円
- リスク加算後原価: 12,567,000 × 1.10 = 13,823,700 円
- 提案価格（粗利35%込み）: 13,823,700 ÷ 0.65 = 約 21,267,231 円
提案価格（丸め）: 21,270,000 円（税別）

参考オプション（別途）
- SSO（Azure AD）: +1.0〜1.5 人月（見積: +1,500,000〜+2,300,000 円 程度）
- 通知（メール/Teams）: +0.6〜1.0 人月
- 外部カレンダー連携（iCal/Google）: +0.8〜1.2 人月

## 7. 前提・制約
- 同時接続/データ量は中規模想定（要件確定時に性能目標を再定義）
- 既存アカウント基盤がない前提（SSOはオプション）
- Azure課金、外部サービス利用料、ドメイン/証明書費用は別途実費
- 運用/保守（月次）は本見積に含まず（別途SLA/料金表をご提案）
