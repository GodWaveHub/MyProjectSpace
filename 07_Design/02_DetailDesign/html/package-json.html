1. ファイル概要

ファイル名package.json
役割Node.jsプロジェクトの設定ファイル。依存関係・スクリプト・メタデータを定義

2. プロジェクト設定

項目名値概要
namegis-map-appプロジェクト名
version0.0.0バージョン番号（開発中）
privatetruenpm公開を禁止
typemoduleESモジュール形式を使用

3. スクリプト

スクリプト名コマンド概要
devvite開発サーバーを起動（ホットリロード有効）
buildtsc -b && vite buildTypeScriptをコンパイルし、本番用ビルドを生成
linteslint .ESLintでコード品質チェックを実行
previewvite previewビルド済みアプリケーションをプレビュー
