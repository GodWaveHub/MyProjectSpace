1. ファイル概要

ファイル名sampleLayer.ts
パッケージsrc/data/
役割アプリケーションの初期表示用サンプルレイヤーデータの定義
依存関係
  • geojson型 (Feature)
  • types/layer.ts (Layer)

2. 処理詳細

2.1 提供するデータ

定数名型説明
defaultLayersLayer[]アプリケーション初回起動時に表示されるサンプルレイヤーの配列

2.2 サンプルフィーチャ

名称ジオメトリタイプ座標説明
HQ (Sample) Point [139.767125, 35.681236] (東京駅付近) プロジェクト拠点のサンプル位置
重点調査エリア Polygon 東京駅周辺の矩形エリア 強調表示するポリゴンの例

2.3 デフォルトレイヤー設定

プロパティ値
id'layer-sample'
name'サンプルレイヤー'
visibletrue
style.strokeColor'#1976d2'
style.fillColor'#90caf9'
style.fillOpacity0.45

3. 入出力仕様

3.1 エクスポート

エクスポート名型説明
defaultLayersLayer[]名前付きエクスポート
defaultLayer[]デフォルトエクスポート（defaultLayersと同じ）

4. 用途

  • アプリケーション初回起動時のデモンストレーション
  • 「サンプル読込」ボタンクリック時の初期データ復元
  • 開発・テスト時の標準データセット
