vite.config.ts - 詳細設計

1. ファイル概要

ファイル名vite.config.ts
役割Viteビルドツールの設定ファイル

2. 設定項目

項目名値概要
plugins[react()]@vitejs/plugin-reactを使用してReact Fast Refreshとjsx変換を有効化

3. プラグイン詳細

@vitejs/plugin-react

  • React Fast Refresh（ホットリロード）のサポート
  • JSX/TSXの自動変換
  • 開発時のエラーオーバーレイ表示

4. 備考

このプロジェクトはシンプルな構成のため、Vite設定は最小限です。必要に応じて以下の設定を追加可能:

  • server.port - 開発サーバーのポート番号
  • build.outDir - ビルド出力ディレクトリ
  • resolve.alias - パスエイリアス
  • define - 環境変数の定義
