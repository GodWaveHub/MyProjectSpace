1. ファイル概要

ファイル名 MapCanvas.tsx
パッケージ src/components/
役割 Google Maps JavaScript APIを使用した地図表示・レイヤー管理・図形作図の中核コンポーネント
依存関係
  • @googlemaps/js-api-loader
  • geojson (Feature, FeatureCollection)
  • react (hooks: useCallback, useEffect, useMemo, useRef, useState)
  • LayerPanel.tsx
  • utils/geojson.ts
  • data/sampleLayer.ts
  • types/layer.ts

2. 処理詳細

2.1 主要な責務

  • Google Maps APIの初期化と地図のレンダリング
  • レイヤーの状態管理（追加・削除・表示切替・スタイル変更）
  • Drawing Managerによる図形作図機能の提供
  • GeoJSONファイルの読み込み・書き出し
  • レイヤー一覧のJSON形式での保存・復元
  • 地図上のフィーチャのクリック・ホバーイベント処理
  • InfoWindowによるフィーチャ情報の表示

2.2 状態管理

状態名 型 説明
layers Layer[] レイヤーの配列。各レイヤーはID、名前、可視性、スタイル、GeoJSONフィーチャを含む
activeLayerId string | null 現在アクティブなレイヤーのID。図形作図時の追加先となる
status Status ユーザーへのメッセージ。type: info/success/error と message を持つ

2.3 主要メソッド

メソッド名 概要
rebuildMapLayers 指定されたレイヤー配列をもとに、地図のDataLayerを再構築する。既存フィーチャを削除後、可視状態のレイヤーのフィーチャを追加
attachDataLayerInteractions DataLayerにクリック・マウスオーバー・マウスアウトのイベントリスナーを設定し、InfoWindow表示やスタイル変更を行う
initializeDrawingManager Google Maps Drawing Managerを初期化し、マーカー・ポリゴン・ライン・矩形・円の作図機能を有効化
handleOverlayComplete Drawing Managerで図形作図完了時に呼ばれ、Google Maps図形をGeoJSON Featureに変換してアクティブレイヤーに追加
handleSaveGeoJson アクティブレイヤーのフィーチャをGeoJSON FeatureCollectionとしてファイル保存
handleFileChange GeoJSONファイルを読み込み、パース・正規化してアクティブレイヤーに追加
handleAddLayer 新規レイヤーを作成し、レイヤー配列に追加してアクティブ化
handleDeleteLayer 指定されたIDのレイヤーを削除
handleToggleLayer 指定されたIDのレイヤーの可視性を切り替え
handleUpdateLayer 指定されたIDのレイヤーの名前またはスタイルを更新
handleSaveLayerList すべてのレイヤー情報をLayerCollectionFile形式のJSONとして保存
handleLoadLayerList LayerCollectionFile形式のJSONを読み込み、レイヤー一覧を復元

3. 入出力仕様

3.1 入力

項目 型 説明
環境変数: VITE_GOOGLE_MAPS_API_KEY string Google Maps APIキー。設定されていない場合はエラーメッセージを表示
GeoJSONファイル File (JSON) Feature, FeatureCollection, またはFeature配列形式のGeoJSONファイル
レイヤー一覧ファイル File (JSON) LayerCollectionFile形式のJSON。version, exportedAt, layers を含む
ユーザー操作 Event 地図上での作図、ファイル選択、ボタンクリックなど

3.2 出力

項目 型 説明
GeoJSONファイル Blob (JSON) アクティブレイヤーのFeatureCollectionをファイルとしてダウンロード
レイヤー一覧ファイル Blob (JSON) 全レイヤー情報をLayerCollectionFile形式でダウンロード
UIレンダリング ReactElement 地図、レイヤーパネル、ツールバー、ステータスバナーを含むUI

4. 例外・バリデーション仕様

条件 処理
APIキーが未設定 エラーメッセージを表示し、地図の初期化を中止
Google Maps API読み込み失敗 エラーメッセージを表示（エラー内容を含む）
アクティブレイヤーが未選択 図形追加・保存・クリア時にエラーメッセージを表示
GeoJSON変換失敗 エラーメッセージを表示し、図形をマップから削除
ファイル読み込み失敗 エラーメッセージを表示し、入力フィールドをリセット
JSONパースエラー エラー内容を含むメッセージを表示
不正なGeoJSON形式 「GeoJSON形式ではありません」とエラー表示
不正なレイヤー一覧ファイル 「レイヤー一覧ファイルの形式が不正です」とエラー表示
