eslint.config.js - 詳細設計

← トップページへ戻る

1. ファイル概要

ファイル名eslint.config.js
役割ESLintコード品質チェックツールの設定ファイル（Flat Config形式）

2. 設定項目

2.1 グローバル無視設定

項目名値概要
globalIgnores['dist']ビルド出力ディレクトリをLint対象から除外

2.2 対象ファイル

項目名値概要
files**/*.{ts,tsx}TypeScript/TSXファイルを対象

2.3 拡張設定

設定名概要
js.configs.recommendedESLint推奨のJavaScript基本ルール
tseslint.configs.recommendedTypeScript用の推奨ルール
reactHooks.configs.flat.recommendedReact Hooks の正しい使用方法を強制
reactRefresh.configs.viteVite + React Fast Refresh 対応ルール

2.4 言語オプション

項目名値概要
ecmaVersion2020対応するECMAScriptバージョン
globalsglobals.browserブラウザ環境のグローバル変数（window, document等）を認識

3. 主要な検出項目

この設定により以下の項目が自動的にチェックされます:

  • JavaScript基本: 未定義変数、到達不能なコード、不要なセミコロン等
  • TypeScript: any型の多用、型アサーションの誤用、型定義の不足等
  • React Hooks: useEffect依存配列の不足、Hooksの呼び出し順序違反等
  • React Refresh: Fast Refreshが正しく動作しないエクスポート形式の検出

4. 備考

ESLint 9.x のFlat Config形式を採用しており、従来の.eslintrc形式より明確で拡張性の高い設定構造を実現しています。
